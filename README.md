# JMeter2Excel

JMeter Test Cases to Excel

## Installation

Clone this repository:

```
$ git clone git@github.com:jinhucheung/jmeter2excel.git
```

And then execute:

```
$ bundle install
```

## Usage

Execute to export excel:

```
$ ruby ./run.rb $jmeter_file_path
```

Then exported excel file in `outs` directory.
